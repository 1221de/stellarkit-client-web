# Stellar Client Web

Code for [`https://stellar.army`](https://stellar.army).

Also provides the contents of the [`stellar-client electron`](https://github.com/StellarKit/stellar-client) app

To build and run:

    npm install
    npm run dev

To build installer

    npm run dist
